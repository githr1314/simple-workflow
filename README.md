# simple-workflow

## 项目介绍

- 基于[开源项目Topology](https://github.com/le5le-com/topology)开发的一个简单流程图demo
- 前端主要分为3部分：
    - 可用的流程图标选择栏；
    - 画布，用于绘制和展示；
    - 属性栏，展示或者设置选中模块的属性；
- 功能：
    - 可以保存绘画的流程图为图片
    - 可以以json的数据格式保存流程图，以数据的方式去分析；

## 项目安装依赖
```
npm install
```

### 开发模式
```
npm run dev
```

### 打包
```
npm run build
```
